# Talk Kindly

Advanced offensive language detection and content moderation platform developed using Python and Natural Language Processing techniques.

## Overview

Talk Kindly is a web based AI content analysis platform designed to detect offensive, harmful, and inappropriate language in text input. The project focuses on lexicon based NLP techniques combined with pattern recognition and masked word detection.

The system was developed as a final year Computer Science project and includes user authentication, analytics, content analysis, file scanning, and administrative features.

---

## Features

- Offensive language detection
- Masked and symbol substituted word recognition
- Real time content analysis
- User authentication system
- User analysis history tracking
- Analytics dashboard
- File analysis support
- Admin panel
- API integration support
- Confidence scoring system
